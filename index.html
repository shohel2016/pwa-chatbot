Machine Troubleshooting Guide

H.V. Leakage

Go to F1 > F6 (H.V. Monitor) and adjust the leakage sensitivity between 0-1000. Default value is 800. If set to 1000, "H.V. Fault" will be disabled.

Viscosity Calibration

Go to F2 > F8 (Ink Maintenance > Viscosity Calibration) and set the reference value (e.g., 3600).

Nozzle Blockage

Go to F3 > F7 (Nozzle Maintenance > Nozzle Cleaning / Unblock). Run the "Nozzle Cleaning" and "Nozzle Unblock" functions. If the issue persists, clean the nozzle using an ultrasonic cleaner.

Ink Prime

Go to F4 > F5 (Ink Maintenance > Ink Prime) and ensure there is no air bubble in the ink system.

Gutter Fault

Go to F5 > F9 (System Maintenance > Special Setup > Gutter Maintenance) and clean the gutter if required.

Pump Speed

Go to F6 > F9 (Manual Control > Pump Speed) and set the pump speed (e.g., 50 RPM).

Temperature Issue

Monitor the temperature directly. Ensure the environment temperature is between 5째C and 45째C. Contact technical support if the issue persists.

Ink Pressure

Go to F7 > F8 (Ink Maintenance > Ink Pressure) and set the ink pressure to 3000 mbar.

Barcode Error

Go to F9 > F3 (Insert Item > Barcode) and input the correct data. Verify the barcode before printing.

Shutdown Issue

Go to F1 > F5 (System Main > Quick Shutdown / Auto Flush). For short-term shutdown, use "Quick Shutdown". For long-term shutdown, use "Auto Flush" multiple times or manually clean the system.
